# Pizza_Sales_Analysis_Dashboard
# 🍕 Pizza Sales Dashboard (Microsoft Excel)

## 📌 Project Overview
This project presents an interactive **Pizza Sales Dashboard** built in **Microsoft Excel** to analyze sales performance and customer purchasing behavior. The dashboard provides key business insights through dynamic visualizations and KPI tracking, enabling data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales performance and revenue.
- Monitor key business KPIs.
- Identify top- and bottom-performing pizzas.
- Understand customer ordering patterns by day, hour, category, and size.
- Support business decisions through interactive dashboards.

## 📊 Key Performance Indicators (KPIs)
- **Total Revenue:** $817,860
- **Total Orders:** 21,350
- **Total Pizzas Sold:** 49,574
- **Average Order Value:** $38.31
- **Average Pizzas per Order:** 2.32

## 📈 Dashboard Insights
- Analyzed over **48,000+ sales records** using interactive Excel dashboards.
- Identified peak sales days, busiest hours, and customer purchasing trends.
- Compared sales across pizza categories and sizes.
- Highlighted top 5 and bottom 5 selling pizzas based on quantity sold.
- Enabled dynamic filtering using slicers for month-wise analysis.

## 📌 Business Insights
- Weekends (Friday and Saturday) recorded the highest number of orders.
- Peak order times were between **12 PM–1 PM** and **5 PM–8 PM**.
- Classic pizzas contributed the highest sales and order volume.
- Large-sized pizzas generated the maximum revenue.
- The dashboard enables quick identification of high-performing products and sales trends.
